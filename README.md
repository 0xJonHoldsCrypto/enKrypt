<!-- GETTING STARTED -->
## Getting Started

To get started make sure you have `nodejs` and `nvm` installed on your system

### Prerequisites

* npm
  ```sh
  nvm install 16
  nvm use 16
  npm install yarn -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/enkryptcom/enKrypt.git
   ```
2. Install NPM packages
   ```sh
   yarn install
   ```
